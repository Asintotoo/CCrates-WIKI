Installation Guide

Follow the steps below to install the CCrates plugin on your Minecraft server.

Step 1: Check Java and Server Requirements

Ensure that your server is running Java 21 and that you're using at least Spigot or Paper for Minecraft 1.21.

Step 2: Download the Plugin

To install the CCrates plugin, download the CCrates.jar file from the following link:

Download CCrates.jar

Step 3: Place the Plugin in Your Server's Plugins Folder

Once the file is downloaded, place the CCrates.jar file into the plugins folder of your Minecraft server.

Step 4: Restart Your Server

Once the plugin is placed in the plugins folder, please perform a full server restart. Do not use the reload command, as this may cause issues with the plugin.
